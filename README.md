\# Secure Smart Traffic - CMS Server



\## Overview



This project implements the Certificate Management System (CMS) component

for a Secure FOTA infrastructure used in IoT traffic systems.



The CMS is responsible for managing device identity, certificate lifecycle,

firmware trust, audit logging, and blockchain-based integrity tracking.
